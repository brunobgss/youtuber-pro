# Prompt para Gemini - Nova Logo Youtuber Pro

Crie uma nova versão do logotipo para o SaaS "Youtuber Pro" com as seguintes especificações aprimoradas:

## CONTEXTO DO PRODUTO
Youtuber Pro é uma plataforma completa de automação para criadores de conteúdo do YouTube que oferece:
- Processamento automatizado de vídeos (edição, TTS, watermarks, overlays)
- Geração inteligente de vídeos a partir de roteiros usando IA
- Geração automática de metadados (títulos, descrições, tags) com inteligência artificial
- Upload automatizado para múltiplos canais do YouTube
- Dashboard completo para gerenciar múltiplos canais simultaneamente

## ANÁLISE DA VERSÃO ANTERIOR
A logo anterior tinha:
- Ícone com play button em quadrado arredondado vermelho
- Texto "Youtuber" em 3D vermelho
- Badge "PRO" em forma oval/pílula
- Efeito tridimensional e brilhante

## REQUISITOS PARA A NOVA VERSÃO

### 1. DIRECIONAMENTO DE DESIGN
**Sugestão: Abordagem mais limpa e moderna**
- Reduzir ou eliminar o efeito 3D extremo (manter sutil se necessário)
- Foco em flat design com toques modernos (gradientes sutis, sombras leves)
- Tipografia mais clean e legível
- Melhor contraste e legibilidade em tamanhos pequenos

### 2. ELEMENTOS CONCEITUAIS (Escolher 1-2)
**Opção A - Automação Inteligente:**
- Play button integrado com símbolos de automação (setas, circuitos, ondas)
- Representar fluxo/processamento contínuo
- Elementos que sugerem IA e tecnologia

**Opção B - Versão Minimalista Premium:**
- Play button estilizado de forma única
- Tipografia customizada para "Youtuber Pro"
- Badge "PRO" mais sutil e elegante
- Espaço negativo criativo

**Opção C - Híbrido Moderno:**
- Play button com overlay de elementos tecnológicos (hexágonos, linhas de conexão)
- Gradiente sutil vermelho-azul ou vermelho-roxo
- Tipografia moderna com letter spacing otimizado

### 3. PALETA DE CORES REFINADA
**Esquema Principal:**
- Vermelho YouTube: #FF0000 (primário)
- Vermelho Escuro: #CC0000 (hover/shadow)
- Azul Tecnológico: #1E88E5 (accent para IA/tech)
- Roxo Moderno (opcional): #7C3AED (para diferenciação)
- Branco: #FFFFFF (texto/contraste)
- Cinza Escuro: #1F2937 (para variações dark)

**Gradientes Sugeridos:**
- Vermelho → Azul (automação + tech)
- Vermelho → Roxo (premium + inovação)
- Vermelho sólido com sombras suaves

### 4. TIPOGRAFIA
**Características:**
- Fonte sans-serif moderna (Inter, Poppins, Montserrat, ou custom)
- "Youtuber" em weight 700-800 (bold/semi-black)
- "Pro" pode ter:
  - Badge oval/pill moderna com cantos mais suaves
  - Uppercase ou title case
  - Diferente cor (azul/roxo) ou gradiente
  - Subscript/superscript sutil
- Letter spacing ajustado para legibilidade
- Line height otimizado

### 5. VARIAÇÕES OBRIGATÓRIAS
Criar as seguintes versões:

**a) Logo Completa (Horizontal)**
- Ícone + Texto "Youtuber Pro"
- Para header/header principal
- Tamanho sugerido: 200px x 60px aprox

**b) Logo Ícone (Apenas símbolo)**
- Apenas o ícone, sem texto
- Para favicon, app icons, thumbnails
- Deve funcionar em 16x16px até 512x512px
- Tamanhos: 32px, 64px, 128px, 256px, 512px

**c) Logo Compacta (Vertical/Stack)**
- Ícone acima, texto abaixo (empilhado)
- Para uso em cards ou espaços verticais

**d) Versão Monocromática**
- Preto/branco ou escala de cinza
- Para impressão e uso em backgrounds coloridos

**e) Versão Dark Mode**
- Otimizada para fundos escuros (#0F0F0F, #1A1A1A)
- Contraste aprimorado

**f) Versão Light Mode**
- Otimizada para fundos claros/brancos
- Contraste aprimorado

### 6. ESPECIFICAÇÕES TÉCNICAS

**Formatos:**
- SVG (vetorial) - preferencial para web
- PNG transparente (256x256, 512x512, 1024x1024)
- PNG com fundo branco (256x256, 512x512)
- PNG com fundo preto (256x256, 512x512)
- ICO/Favicon (32x32, 64x64)

**Responsividade:**
- Deve funcionar perfeitamente em 32x32px (favicon)
- Escalável sem perda de qualidade
- Detalhes visíveis em pequenos tamanhos
- Simplificado para favicon se necessário

**Área de Segurança:**
- Mínimo 10-15% de espaço ao redor dos elementos
- Elementos não devem tocar as bordas

### 7. DIFERENCIAÇÃO E INOVAÇÃO

**O que deve transmitir:**
- Profissionalismo e confiança
- Automação e eficiência
- Tecnologia avançada (IA)
- Facilidade de uso
- Modernidade

**Evitar:**
- Logos muito similares ao YouTube oficial
- Efeitos 3D excessivos que não funcionam em tamanhos pequenos
- Elementos muito complexos que perdem detalhes ao reduzir
- Cores muito saturadas que causam fadiga visual

### 8. PROCESSO DE CRIAÇÃO

**Por favor, forneça:**
1. **3-5 conceitos diferentes** com explicação da abordagem
2. Para cada conceito:
   - Justificativa das escolhas de design
   - Como representa automação/IA/tecnologia
   - Adaptações para diferentes tamanhos
3. Versões nas variações solicitadas acima
4. Paleta de cores usada com códigos hexadecimais
5. Sugestões de uso (onde aplicar cada variação)

**Critérios de avaliação:**
- ✅ Legibilidade em tamanhos pequenos
- ✅ Escalabilidade sem perda de qualidade
- ✅ Memorabilidade e diferenciação
- ✅ Representa bem automação + IA + vídeo
- ✅ Funciona em claro e escuro
- ✅ Profissional e moderno
